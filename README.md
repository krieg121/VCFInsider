## ğŸ”§ Local Development

### Prerequisites

- Ruby 3.4+
- Bundler gem
- Git

### Setup

1. **Clone the repository**
   ```bash
   git clone https://github.com/YOURUSERNAME/vcf-insider-blog.git
   cd vcf-insider-blog
   ```

2. **Install dependencies**
   ```bash
   bundle install
   ```

3. **Serve locally**
   ```bash
   bundle exec jekyll serve
   ```

4. **Visit** `http://localhost:4000`
